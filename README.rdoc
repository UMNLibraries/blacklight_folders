== How to add blacklight_folders to your blacklight application

* Create a new rails application

* Add the <code>blacklight</code> gem and the <code>blacklight_folders</code> gem to your Gemfile, and run <code>bundle install</code>

* Run the blacklight generator with the "devise" switch, following the {instructions on the blacklight wiki}[https://github.com/projectblacklight/blacklight/wiki/Quickstart#creating-a-new-application-the-hard-way] to set up your blacklight app.  _NOTE:_ depending on your use, you may also want to include the \-\-marc and \-\-jettywrapper + switches.

    $ rails g blacklight:install --devise

* Run the blacklight_folders generator

    $ rails g blacklight_folders:install

---

== Developer Notes

For development on the blacklight_folders gem itself.

==== Install Jetty

  $ rake jetty:unzip
  $ rake jetty:add_test_core
  $ rake jetty:start

==== Generate a test app and run the specs

  $ rake jetty:start
  $ rake engine_cart:clean
  $ rake engine_cart:generate
  $ rake spec

==== Run the rails server for the generated app

  $ rake engine_cart:generate
  $ cd spec/internal
  $ rails s

==== Create test data for your generated app

  $ cd spec/internal
  $ rake blacklight:solr:seed
